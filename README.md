this is a collection of all my dotfiles

a few notes:
in order to make certain commands work with keybinds, it is necessary (at least the way i did it, dont know if theres a better way to do it) to pass along your sudo password with the command.
for security reasons my password has been replaced by PASSWORD, and will need to be replaced in the following files:
- .scripts/bup.sh
- .scripts/bdwn.sh 
these controll screen brightness
- startup.sh
launches some apps in specific workspaces
- hyprland.conf
self exlanatory

some required packages:
- light (allows for brightness control)
- pavucontrol (allows audio control via pulseaudio)
